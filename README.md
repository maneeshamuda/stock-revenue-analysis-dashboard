# stock-revenue-analysis-dashboard
Python project analyzing historical stock price and revenue data using yfinance, BeautifulSoup, pandas, and visualizations.
# Historical Stock & Revenue Data Analysis Dashboard

## Overview
This project analyzes historical stock price and revenue data using Python.

## Companies Analyzed
- Tesla
- GameStop

## Tools Used
- Python
- Pandas
- yfinance
- BeautifulSoup
- Requests
- Matplotlib

## Project Tasks
- Extracted historical stock data using yfinance
- Scraped quarterly revenue data using BeautifulSoup
- Cleaned and transformed datasets using pandas
- Visualized stock price and revenue trends

## Key Learnings
- Data extraction
- Web scraping
- Data cleaning
- Time-series analysis
- Data visualization
